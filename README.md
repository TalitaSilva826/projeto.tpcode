# projeto.tpcode  
Criamos sites modernos, responsivos e estratégicos para posicionar sua marca no ambiente digital.  (empresa ficticia)

Atualizações serão feitas! 
Daremos os últimos toques no frontend, para melhorar a responsividade para mobile e outras interfaces e alguns detalhes estéticos também!
Vamos também incorporar backend ao código para trazer funcionalidade ao botão de "Comece um projeto personalizado", ao apertar o botão o usuário
será direcionado para um formulário, onde irá preencher algumas informações referente a sua empresa e como deseja o projeto do seu site, ao cliar 
em enviar um email será lançado para nós solicitando o orçamento!
